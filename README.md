Shipment Tracking System (Frontend)

A modern full-stack shipment tracking system built using React and
Tailwind CSS. This application allows users to create, track, and manage
shipments with a clean dashboard and real-time tracking interface.

Live Demo:[](logistic-shipment-frontend.vercel.app)

Backend API: https://logistics-backend-pqlo.onrender.com

Features: - User login with JWT authentication - Protected routes -
Create and manage shipments - Shipment tracking with timeline -
Amazon-style progress stepper UI - Role-based access (admin & user
simulation) - Axios interceptor for token handling

Tech Stack: - React.js - Tailwind CSS - Axios - React Router DOM

Setup Instructions: 1. Clone repository 2. Run: npm install 3. Run: npm
run dev

Environment Variable:
VITE_API_URL=https://logistics-backend-pqlo.onrender.com
